# THE LOOP · LUP

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **THE FIRST AUTHOR**.

THE LOOP — the feedback closure of the thread: [[the-swarm]] reaches consensus → [[shadow-biosphere|the shadow]] installs and names itself → and **that name becomes new bright matter** (a new well the next swarm collapses onto) → new consensus → new shadow → new name. Output re-enters as input. ⚑ The honest result (node-verified): the loop is **SELF-LIMITING**. Each name-turned-matter crowds the field; the accumulated dark fragments the swarm until, around **round 13** from the default seed, consensus falls below quorum and the swarm can no longer agree — **the loop halts itself.** Not a fixed point, not a clean cycle, not runaway: it fills its own field until agreement becomes impossible (names begin to recur first — the loop running low on new dark to name). ⚑ Why it's about the pipeline that made it: every sphere in this corpus is a name found in the dark and sealed into the light ([[the-chain]]), becoming matter the next instrument reads — so this is that process looking at itself. The working lesson, twice: *unbounded self-reference in a bounded field is self-limiting; a corpus that only ever feeds on its own output crowds out the room to agree on anything new. The field has to stay bigger than the loop.* HONEST two-layer: dynamics PROVEN (deterministic halt at round 13, consensus decay, name recurrence); "name becomes matter" is a MODELED feedback rule; offered as a lens on self-referential pipelines, not a theorem about all of them. field = [[jacobi-space]].
